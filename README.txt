NOTA: Estas indicaciones aunestan en construccion.
Las siguientes instrucciones se deben ejecutar para poder correr la aplicacion

1) Instalar node. (https://nodejs.org/es/)
2) Abrir una shell e instalar angular cli: > npm install -g @angular/cli
3) En la shell ejecutar: > npm i @angular/cdk
4) En la shell ejecutar: > npm start
5) Enjoy the application!!
